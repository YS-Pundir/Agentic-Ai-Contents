Q1: Which of the following is true about a primary key?

A. It can contain duplicate values
B. It can be null for every row
C. It uniquely identifies a record
D. It stores images

Answer: C

Q2: What is a foreign key?

A. A key used to unlock the database server room
B. A column that refers to the primary key of another table
C. A random value used in Excel
D. A password column

Answer: B

Q3: Which type of relationship usually needs a junction table?

A. One-to-One
B. One-to-Many
C. Many-to-Many
D. Zero-to-One

Answer: C

Q4: Which SQL clause is used to filter rows based on a condition?

A. ORDER BY
B. WHERE
C. GROUP
D. TABLE

Answer: B

Q5: Which SQL clause is used to sort query results?

A. FILTER BY
B. SORT
C. ORDER BY
D. ARRANGE BY

Answer: C

Q6: Which of the following is structured data?

A. Customer profile table
B. Audio recording
C. Image file
D. Video stream

Answer: A

Q7: Which of the following best explains the role of SQL in databases?

A. It is used to design logos
B. It is used to interact with relational databases

Q8: Why are databases preferred over plain files in large AI applications?
