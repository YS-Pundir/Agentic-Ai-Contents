lecture ID: 153013

Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Introduction to LangChain: Concepts, Chains, and Prompt Templates

objective: Understand what LangChain is and build a first reusable prompt-and-chain workflow without framework overload

topics be covered:
* What is LangChain — definition, purpose, and the problem it solves
* Why a framework instead of raw LLM API calls — composability and reuse
* Where LangChain fits in the agentic stack (between the LLM and your application)
* The LangChain mental model — Runnables, chains, and composable pipelines
* LangChain modular architecture: chains, agents, tools, memory, and retrievers at overview level
* LangChain Core and Community packages at a practical level
* PromptTemplate and ChatPromptTemplate
* LCEL (LangChain Expression Language) and the pipe operator for a first chain
* String output handling for predictable model responses
* Instructor walkthrough of a simple PromptTemplate → LLM chain


detailed subtopics to be covered:

* Define LangChain in one sentence — an open-source framework for building applications powered by LLMs;
* Explain why raw LLM API calls become hard to maintain when prompts, memory, tools, retrieval, and chaining are needed;
* Place LangChain on the agentic stack between the model provider and the application;
* Introduce the LangChain mental model — Runnables composed into chains;
* Explain how chains, agents, tools, memory, and retrievers fit together in a single-agent workflow;
* Describe how PromptTemplate and ChatPromptTemplate keep prompts reusable and variable-driven;
* Build or follow a first LCEL chain as PromptTemplate → LLM with string output handling;
* Compare hardcoded prompts vs templated prompts on the same task;
* Note where broader framework choices were introduced in Module 1 and defer multi-agent framework comparison to Module 4
