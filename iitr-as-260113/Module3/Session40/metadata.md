title: LangChain RAG Pipeline

objective: Express prior RAG knowledge as a LangChain retrieval-augmented answering pipeline with evaluable grounding behavior.

type of session: theory/implementation/mixture of theory + implementation / You take the call

topics be covered: loaders; Chroma; retriever; LCEL RAG chain; grounding comparison


detailed subtopics to be covered:
* Ingest and segment a document corpus using LangChain loaders and chunking aligned with the established Module 2 corpus design.
* Embed and persist vectors in Chroma through LangChain so retrieval remains reproducible across runs.
* Construct an LCEL retrieval-augmented answering chain that conditions generation on retrieved passages.
* Critique grounding quality by contrasting outputs with versus without retrieval on representative queries using instructor-supplied comparison criteria.
