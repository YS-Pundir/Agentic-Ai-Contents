lecture ID: 153012

Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Ollama: Exploring Another World of LLMs

objective: Set up and run LLMs locally or through Ollama Cloud so every learner has a working, cost-free LLM setup before building agents

topics be covered:
* LLMs can be run locally on your own machine
* Install Ollama and verify the setup
* Pull and run a light model (0.5B, 1B, and other small parameter models)
* Avoid heavy local models on regular laptops
* Trade-offs of very small local models (limited training data, higher hallucination risk)
* Call a local model from Python with the Ollama API
* Switch to Ollama Cloud when stronger models are needed
* Create an Ollama account and generate a free-tier Ollama API key
* Use the same Python script for local and cloud by changing the API key and host
* Main Ollama capabilities — chat, text generation, embeddings, and more
* Model families available through Ollama — text, vision, audio, and video


detailed subtopics to be covered:
* Understand that LLMs can run locally on your own machine, not only through paid cloud services;
* Install Ollama on your operating system and verify that the setup works;
* Pull a light local model (for example 0.5B or 1B parameter models) and run a simple prompt from the CLI;
* Explain why learners should avoid pulling heavy models on regular laptops;
* Describe the trade-offs of very small local models, including limited training data and a higher chance of hallucination;
* Write a short Python script that calls a local model through the Ollama API and prints the response;
* Create an Ollama account and generate a free-tier Ollama API key;
* Update the same Python script to call Ollama Cloud by changing the API key and host settings;
* Compare local light-model output with Ollama Cloud output on the same prompt;
* List the main things Ollama can do, including chat, text generation, and embeddings;
* Identify the main model types available through Ollama, including text, vision, audio, and video;
* Finish the session with one working Python script that can call either local Ollama or Ollama Cloud
