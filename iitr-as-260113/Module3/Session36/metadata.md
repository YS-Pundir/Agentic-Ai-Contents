Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 1hr 45 mins

title: LangChain Environment Setup and First LCEL Chain

objective: Stand up the LangChain project environment and prove a first LCEL chain end-to-end on Ollama.

topics be covered: venv; packages; .env; folders; ChatOllama; ChatPromptTemplate; LCEL; StrOutputParser; hello_chain.py


detailed subtopics to be covered:
* Construct an isolated Python environment with LangChain and Ollama integration dependencies suitable for the module worktree.
* Apply secure project layout and environment-variable conventions for collaborative LangChain development.
* Demonstrate correct model binding using ChatOllama with credentials and endpoints from the Ollama setup block.
* Compose and execute a minimal LCEL pipeline combining templated prompts chat model and deterministic string output parsing.
* Validate the first chain behaves consistently across distinct inputs using instructor-defined success criteria.
