Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 1hr 50 mins

title: Agent Tool Use

objective:  Enable function calling so agents invoke tools with structured arguments and results.	

topics be covered: Function calling; tool schemas; execution cycle; tool result handling


detailed subtopics to be covered:
* Describe tools in schema form so the model can select and parameterize calls.
* Register at least one callable tool and bind it to the agent or graph executor.
* Execute the model-tool loop: propose call; run function; return result to the model.
* Verify tool outputs are fed back correctly before the next reasoning step.
