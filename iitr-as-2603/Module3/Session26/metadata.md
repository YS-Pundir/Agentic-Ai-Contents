title: Working with APIs and JSON

objective: Call external HTTP APIs and parse JSON so agents can use real data sources.

type of session: theory/implementation/mixture of theory + implementation / You take the call

topics be covered: REST APIs; HTTP methods; JSON structures; API responses; requests library (lite)


detailed subtopics to be covered:
* Explain how client applications communicate with REST APIs using HTTP methods and status codes.
* Read and write JSON structures that represent requests and responses in Python.
* Execute a GET request with the requests library and handle common response errors safely.
* Map JSON fields from an API response into variables used in a downstream LLM prompt.
