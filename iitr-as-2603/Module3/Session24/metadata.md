Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 1hr  50mins

title: Mastering Prompt Engineering

objective: Design prompts that reliably steer model behavior for beginner agent tasks.	

type of session: theory/implementation/mixture of theory + implementation / You take the call

topics be covered: System vs user prompts; zero-shot and few-shot; chain-of-thought; prompt templates; agent prompt design (lite)


detailed subtopics to be covered:
* Distinguish system and user roles and write a clear system prompt for a bounded task.
* Apply zero-shot and few-shot examples to improve answer format and consistency.
* Use chain-of-thought prompting for multi-step reasoning on a structured problem.
* Assemble reusable prompt templates suitable for a single-agent workflow.
