Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Data Prep: Handling Messy Data	

objective: Advanced cleaning for machine learning.
 
topics be covered: Imputing missing values; Removing duplicates; Categorical encoding; Scaling and Normalization


detailed subtopics to be covered:

- Load and Inspect a Dataset — Open the dataset in Google Colab, explore rows and columns, and identify issues like missing values, duplicates, inconsistent data, and obvious outliers;
- Handle Missing Data — Detect missing values and apply imputation techniques such as mean, median, and mode based on the type of data;
- Remove Duplicates and Ensure Consistency — Identify duplicate records and clean inconsistencies or unrealistic values (outliers) to maintain data accuracy;
- Prepare Categorical Data — Identify categorical columns and understand the need to convert them into numerical format;
- Apply Encoding Techniques — Implement label encoding and one-hot encoding on categorical variables and observe how the dataset changes;
- Apply Feature Scaling — Normalize and standardize numerical features to ensure uniform scale across all inputs;
- Prevent Data Leakage in Preprocessing — Apply cleaning steps on training data and reuse them on test data to maintain correct evaluation;
- Evaluate the Impact of Data Preparation — Compare dataset before and after preprocessing to understand improvements in quality and readiness for modelling;
