Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: The ML Workflow & Habits

objective: Learn the standard lifecycle of an ML project.

topics be covered: Problem framing; Features vs Labels; Training/Validation/Test split; Baselines


detailed subtopics to be covered:

Introduce Google Colab Notebook — Set up and navigate Google Colab, understand its interface, and run basic code cells for ML workflows.
Define the ML Workflow — Explain the end-to-end lifecycle of an ML project and why a structured approach matters.
Frame ML Problems Effectively — Translate real-world problems into ML terms by identifying objectives, inputs, and expected outputs.
Differentiate Features and Labels — Identify and distinguish between input variables (features) and target variables (labels) with examples.
Apply Data Splitting Strategies — Explain and implement training, validation, and test splits while avoiding data leakage.
Establish and Evaluate Baselines — Define simple baseline models and compare them with model outputs to measure improvement.
