| SQL Basics – tables rows and columns understanding datasets | Covered | Relational DB motivation vs Excel/CSV; tabular model, rows/columns, schemas, primary keys; SQL as interface to stored tables; hands-on employee table context. |

| Sorting and Limiting – ORDER BY and LIMIT | Covered | `ORDER BY` ASC/DESC, `LIMIT` for top-N; combined with `SELECT`/`FROM`/`WHERE`; execution order of clauses discussed. |
| Pandas vs SQL – mapping dataframe selection and filtering to SQL queries | Covered | Repeated mapping: `display`/`select *`, filtering vs `WHERE`, `sort_values` vs `ORDER BY`, `head(n)` vs `LIMIT`; side-by-side SQL vs pandas slide; live pandas snippets where relevant. |
