title: Strings, Dictionaries & Classes	

objective: Work with text, structured data, and basic object-oriented concepts	

topics to be covered: string operations;dictionaries; inbuilt functions; basics of classes

detailed subtopics/Learning Objectives:
- Implement string data types for text
- Create and initialize dictionaries for key-value pairs
- Access dictionary values using keys and get() method
- Add and modify dictionary entries
- Remove entries using pop(), popitem(), and del
- Check for key existence using the 'in' keyword
- Iterate through dictionary keys, values, and items
- Merge dictionaries using the update() method
- Other Inbuilt functions of list
- Define classes as object blueprints
- Create objects from class definitions
- Implement __init__ constructor for initialization


Session Duration: 2hr 15 mins
