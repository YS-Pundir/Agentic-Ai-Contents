Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Problem Solving Session: Plotting & Storytelling with Data

objective: Practice a medium-level hands-on visualisation workflow by loading a simple dataset, creating basic charts, and writing short insights that can be completed within 2 hours

topics be covered:


detailed subtopics to be covered:
- Dataset Loading and Quick Inspection – import pandas, read one CSV file, check shape, preview rows and understand the main columns; 
- Basic Data Readiness Check – check missing values and data types, and make only simple fixes required before plotting; 
- Simple Summary Analysis – use value counts and basic groupby summaries to identify 2-3 useful observations from the dataset; 
- Matplotlib Practice – create one bar chart for category comparison, one line chart for trend and one histogram for distribution; 
- Basic Plotly Practice – recreate one important chart as an interactive Plotly chart with hover details; 
- Chart Selection Practice – choose the correct chart for simple questions like comparison, trend and distribution; 
- Chart Labelling and Formatting – add clear titles, axis labels, colours and layout adjustments to make charts readable; 
- Short Insight Writing – write 1-2 line observations below each chart explaining what the chart shows; 
- Final Mini Output – complete a small notebook-style solution with cleaned data, 3-4 charts and a short final summary
