
### Q1. What is the main purpose of a Version Control System (VCS)?

A. To delete old files permanently
B. To track and manage changes in project files over time
C. To compile code automatically
D. To increase internet speed

**Answer: B**

---

### Q2. Which problem is solved by using version control?

A. Slow internet connection
B. Loss of code history and overwriting changes
C. High CPU usage
D. Syntax errors in code

**Answer: B**

---

### Q3. In Git, where are changes permanently stored as snapshots?

A. Working Directory
B. Staging Area
C. Local Repository
D. Remote Server

**Answer: C**

---

### Q4. Which command is used to check the current state of files in Git?

A. git log
B. git status
C. git diff
D. git branch

**Answer: B**

---

### Q5. What is the purpose of the staging area in Git?

A. To delete files permanently
B. To prepare selected changes before committing
C. To push code to GitHub
D. To store backups online

**Answer: B**

---

### Q6. Which command creates a new commit in Git?

A. git add
B. git commit
C. git push
D. git clone

**Answer: B**

---

### Q7. What is the key difference between Git and GitHub?

A. Git is cloud-based, GitHub is local
B. GitHub is a command-line tool, Git is a website
C. Git is a local version control tool, GitHub is a cloud hosting platform
D. Both are exactly the same

**Answer: C**

---

### Q8. What is a branch in Git?

A. A backup file
B. A type of commit message
C. An independent line of development
D. A deleted repository

**Answer: C**
