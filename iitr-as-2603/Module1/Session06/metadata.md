title: Version Control and GitHub Basics	

objective: Understand how version control helps manage code changes and collaborate using Git and GitHub	

topics to be covered: Version Control Basics – tracking changes and maintaining code history; Git Workflow – cloning repositories and managing local code copies; Core Commands – add commit and push for saving changes to repositories; GitHub Repositories – syncing local work with remote repositories; Collaboration Concepts – overview of branching merging and resolving conflicts


Deatailed Subtopic/Learning Objectives:
- Describe Version Control
- Install & configure Git identity
- Track & manage file changes
- Undo mistakes safely 
- Develop features using branches
- Configure & synchronize remote repositories
- Collaborate using PRs & repository access control


Session Duration: 2hr 15 mins
