
### Q1. What is the main purpose of using loops in programming?

A. To reduce memory usage only
B. To repeat a block of code multiple times
C. To store data permanently
D. To replace functions

**Answer:** B

---

### Q2. Which type of loop is best when the number of iterations is unknown?

A. for loop
B. while loop
C. range loop
D. nested loop

**Answer:** B

---

### Q3. What will happen if the condition in a while loop never becomes False?

A. The loop runs only once
B. The program crashes immediately
C. The loop becomes infinite
D. The loop skips execution

**Answer:** C

---

### Q4. Which loop is most suitable for iterating over a list or string?

A. while loop
B. for loop
C. do-while loop
D. infinite loop

**Answer:** B

---

### Q5. What is the output of the following code?

```python
for i in range(3):
    print(i)
```

A. 1 2 3
B. 0 1 2
C. 0 1 2 3
D. 1 2

**Answer:** B

---

### Q6. What does the `break` statement do in a loop?

A. Skips current iteration
B. Restarts the loop
C. Exits the loop immediately
D. Pauses execution

**Answer:** C

---

### Q7. What does the `continue` statement do?

A. Stops the loop completely
B. Skips the current iteration and moves to the next
C. Ends the program
D. Repeats the same iteration

**Answer:** B

---

### Q8. In nested loops, how many times does the inner loop execute?

A. Only once
B. Equal to outer loop
C. Completely for each iteration of outer loop
D. Random number of times

**Answer:** C
