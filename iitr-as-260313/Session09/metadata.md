Course name: Certifiction in agentic systems and design 

target audience: students from any background not necessarily with tech background

title: Prompt Engineering, Business Use Cases & Framework Overview

objective: Learn how prompting, use cases, and frameworks support building AI agents


topics to be covered: system prompts;role prompting;agent use cases;frameworks overview:;LangChain;CrewAI;AutoGen


detailed subtopics to be covered:

* Understand System Prompts — Learn how initial instructions define agent behavior, tone, and boundaries;
* Apply Role Prompting — Use roles like “teacher”, “assistant”, or “analyst” to guide responses;
* Write Simple Effective Prompts — Learn how to give clear instructions for better outputs;
* Control Output Format — Guide responses (e.g., bullet points, steps, short answers);
* Experiment with Prompt Variations — See how small changes in prompts change outputs;
* Explore Real-World Use Cases — Identify applications like support bots, coding help, and research assistants;
* Map Use Case to Agent Behavior — Understand how different problems require different agent approaches;
* Understand Why Frameworks Exist — Learn why tools are needed to build agents easily;
* Explore LangChain — Understand its role in connecting models, tools, and workflows (high level);
* Explore CrewAI — Learn how multiple agents can work together with defined roles;
* Explore AutoGen — Understand agent-to-agent communication conceptually;
* Compare Frameworks (Simple) — Identify when to use which framework without going deep into code;
* Design a Simple Agent Flow — Combine prompt + LLM + action in a basic conceptual workflow;
* Do a Guided Mini Activity — Create a simple role-based agent using prompts (no heavy coding);
