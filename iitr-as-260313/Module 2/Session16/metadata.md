- Differentiate Keyword Search vs Semantic Search — When each approach wins; compare exact-match and meaning-based retrieval;
- Understand Semantic Search Workflow — Store content, embed documents, embed query, compare vectors, return closest matches; defer indexing, ANN, and vector DB implementation to Session 17;
- Explore Applications and Embeddings in Agentic Systems — Semantic search, recommendations, conversational AI, RAG retrieve step, and agent memory or context retrieval by meaning;
