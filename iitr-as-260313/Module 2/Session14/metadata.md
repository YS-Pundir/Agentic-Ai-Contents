lecture ID: 152510

Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Introduction to Databases for AI Systems

objective: Understand how databases store and organize data used by AI systems

topics be covered: relational databases;SQL basics;structured vs unstructured data


detailed subtopics to be covered:
- Understand the Need for Databases — Explain why file-based storage (CSV/Excel) breaks down at scale, and what problems databases solve: scalability, consistency, concurrent access, and queryability in AI systems;
- Explore Types of Databases and Their Use Cases — Introduce Relational (SQL), NoSQL (document/key-value), Vector Databases, and Time-Series databases; explain when each is used in AI applications;
- Learn Core SQL Terminologies — Define tables, rows, columns, schemas, primary keys, foreign keys, constraints, and data types using visual real-world analogies;
- Explore Where SQL Can Be Written — Instructor walkthrough of MySQL and Supabase; highlight Supabase as the go-to tool (no installation needed, browser-based, and easily connects to AI systems in future sessions);
- Connect and Explore the Supabase Interface — Navigate the Supabase dashboard, create a project, and understand how to interact with the database through the SQL editor and table UI;
- Create Tables and Insert Data (CREATE & INSERT) — Design a simple table schema representing a real-world entity and insert records using Supabase's SQL editor;
- Read Data with SELECT Queries — Write SELECT queries to retrieve all rows and specific columns; understand the basic anatomy of a SQL query;
- Update and Delete Records (UPDATE & DELETE) — Modify existing records and remove rows; understand the critical role of the WHERE clause in write operations to avoid accidental data loss;
- Write Queries with Filtering — Use WHERE clause with comparison operators, logical operators (AND, OR, NOT), and pattern matching (LIKE, IN, BETWEEN) to retrieve precise records;
- Differentiate Structured vs Unstructured Data — Compare tabular data with text, images, and audio, and explain their relevance in AI systems;
