lecture ID: 148083

Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Short-Term vs Long-Term Memory

objective: Learn different memory types and storage approaches for agents

topics be covered: conversational memory;persistent memory;memory storage strategies; (Main focus on Therotical discussion)	


detailed subtopics to be covered:
- Recall the Role of Memory in Agent Behavior — Revisit how memory enables stateful interactions, continuity, and improved decision-making in agents compared to stateless systems;
- Differentiate Short-Term and Long-Term Memory in Agents — Clearly define both memory types and compare them based on duration, storage, and their role in agent behavior;
- Explain How Conversation History Functions as Short-Term Memory — Describe how the context window and ongoing interaction history act as working memory for agents during a session;
- Analyze the Limitations of Context-Window-Based Memory — Highlight constraints such as token limits, increasing cost, and performance degradation in long conversations;
- Design Short-Term Memory Strategies (Buffer, Window, Summary) — Explain how different approaches manage conversation history and when to use each strategy based on practical needs;
- Explain Long-Term Memory and Its Types (Episodic, Semantic, Procedural) — Define different forms of persistent memory and how agents store and utilize past interactions, knowledge, and action patterns;
- Implement Basic Memory Strategies in Agent Workflows — Demonstrate how short-term memory strategies can be integrated into an agent pipeline using simple logic or pseudo-implementation;
- Evaluate How Agents Decide What to Remember and Introduce Storage Awareness — Discuss how agents prioritize relevance vs recency, and briefly introduce external storage concepts as a foundation for upcoming sessions;
