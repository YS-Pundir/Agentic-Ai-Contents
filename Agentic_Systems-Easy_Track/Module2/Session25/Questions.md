### Q5. In a machine learning context, what are “features”?

A. The final predictions of a model
B. The input variables used to make predictions
C. The errors in the dataset
D. The evaluation metrics

### Q6. In a machine learning problem, what is a “label”?

A. A column used only for visualization
B. The input data

### Q7. Which of the following best describes rule-based AI?

A. Systems that learn patterns automatically from data
B. Systems that rely on randomly generated outputs
C. Systems that follow explicitly programmed rules and logic
D. Systems that require no programming

**Answer:** C
