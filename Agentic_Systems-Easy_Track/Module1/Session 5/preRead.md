## Introduction: How Programs Decide
Think about deciding what to wear.

- If it’s raining → take an umbrella  
- Else → don’t  

You follow conditions every day without thinking about them.

Programs do the same thing—but they need the rules written clearly.  
These rules are called **conditionals**.

---

## Why Logic Matters
Without logic, programs can only run straight from top to bottom.

Logic allows programs to:
1. **React to different situations**  
2. **Choose between actions**  
3. **Handle real-world uncertainty**  

Every smart program—AI or not—depends on this.

---

## From Known to New: Choices → If-Else
### Everyday Thinking
You already think in conditions:
- If battery is low, charge phone  
- If food is hot, wait  

### Python Thinking
Python uses **if**, **else**, and **elif** to express these choices.

The computer checks a condition and decides which path to follow.

---

## Core Concepts for Program Logic
### 1. Conditionals (If-Else)
Conditionals let a program run different code based on a condition.

- **if** checks a condition  
- **else** runs when the condition is false  

Only one path runs at a time.

---

### 2. Boolean Values
At the heart of every decision is a **Boolean**.

- **True**  
- **False**  

Every condition in Python eventually becomes one of these two values.

---

### 3. Logical Operators
Logical operators combine or change conditions.

- **and** → both conditions must be true  
- **or** → at least one condition must be true  
- **not** → reverses a condition  

They allow more complex decisions without writing many if statements.

---

### 4. Indentation Rules
Python uses **indentation** to define blocks of code.

- Indented code belongs to a condition  
- Misplaced indentation changes program meaning  

*Indentation is not styling—it is logic.*

---

## Putting It All Together
A program might:
1. Take user input  
2. Check multiple conditions using logical operators  
3. Decide which block of code to run  
4. Execute only the correctly indented section  

This is how programs “think.”
